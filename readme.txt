This Plugin makes Geocache Bloggers life easyer, with a little Waypoint linking shortcut.

== Description ==

This Plugin adds a shortcut to Wordpress for the Geocache redirection Service [coord.ch](http://coord.ch/).

[Coord.ch](http://coord.ch/) supports currently the following Geocache Listing Sites:

*   GC - geocaching.com
*   PR - geocaching.com Profillink
*   WM - waymarking.com
*   OC - opencaching.de Deutschland
*   OZ - opencaching.cz Tschechien
*   OJ - opencaching.jp Japan
*   ON - opencaching.no Norwegen
*   OK - opencaching.org.uk Grossbritannien
*   OP - opencaching.pl Polen
*   OS - opencaching.se Schweden
*   OU - opencaching.us United States
*   OX - opencaching.com Garmin
*   TC - terracaching.com
*   GA - GeoCaching Australia
*   N  - navicache.com

[Coord.ch](http://coord.ch/) supports currently the following trackables:

*   TB - Geocaching.com travelbugs and coins
*   GK - GeoKrety.org

== Installation ==

1. Upload the `coordch-geocaching-shortcut` Folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I use this shortcut? =

Very simple, add the following tag: `[wp id=GC77 text="Your Desciption"]`.

Alternativly click in the WYSIWYG editor and add this tag as URL: `[wp id=GC77]`.

= Can I set a title tag? =

Yes, `[wp id=GC77 text="Your Desciption" title="Your title tag"]`.

In the WYSIWYG editor simply fill the title field.
